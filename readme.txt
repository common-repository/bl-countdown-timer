Plugin to manage multiple countdown timers on your single or multi-site wordpress installation.


== Description ==

Our plugin is used to setup and manage multiple countdown timers on your wordpress site. It comes with an easy to use admin panel so you can add, edit and delete countdown timers from within Wordpress.

If you want to temporarily remove a countdown timer without deleting the details then you can disable it and re-enable when your ready.

The plugin comes with it's own widget so you can place your countdown timer into one or multiple sidebar widget areas on your site.


== Installation ==

1. Upload the pluign to the plugins directory `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Appearance -> Widgets menu in WordPress and drag/drop the Countdown Timer widget into one of your sidebars in WordPress.
